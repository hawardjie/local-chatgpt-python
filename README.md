# local-chatgpt-python

Local ChatGPT LLM using python and openAPI

# Required setup

## Install Ollama

https://ollama.com/

## Test ollama

```bash
$ ollama run llama3.2
>>> Teach me how to run a small business
...
Use Ctrl + d or /bye to exit.
>>> /bye
```

## Install Requirements

```bash
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```
